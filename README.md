#  MyBills+ - Personal Finance & Bill Tracker

MyBills+ is a modern Android application designed to help users manage their bills, subscriptions, and personal finances in a simple and efficient way.

##  Features

-  Smart bill reminders
-  Optional PIN security
-  Subscription tracking
-  Dark / Light mode support
-  Ad-free experience (Premium)
-  Fast and intuitive user interface
-  Multi-language support

##  Technologies Used

- Kotlin
- Android Studio
- Jetpack Compose
- Material Design 3
- DataStore
- Google Play Billing
